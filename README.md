# Built Not Born — Inshape Mind

Interaktiver Blueprint (React + Vite) für die Inshape Mind Marke.

## Lokal testen (optional)

```bash
npm install
npm run dev
```

## Deploy auf Vercel

1. Dieses Projekt als neues Repository auf GitHub hochladen (z.B. "built-not-born").
2. Auf vercel.com mit GitHub einloggen.
3. "Add New Project" → das Repo auswählen.
4. Vercel erkennt automatisch "Vite" als Framework — Einstellungen so übernehmen.
5. "Deploy" klicken. Nach 1–2 Minuten ist die App live unter einer URL wie
   `built-not-born.vercel.app`.

## Updates später

Jede Änderung, die du in das GitHub-Repo pushst (z.B. neue Texte, Farben),
wird automatisch von Vercel neu deployed — kein manuelles erneutes Hochladen nötig.

## Struktur

- `src/Blueprint.jsx` — die komplette App (Inhalte, Design, Rechner, Sprachen)
- `src/main.jsx` — Einstiegspunkt
- `index.html` — Grundgerüst, Fonts, Meta-Tags
